LATEST

The goal of this website is to showcase the love I have for the community that I'm from-South San Diego right along the border with Tijuana, MX. Kevin Bacon 6 degrees of freedom graph search

Kevin Bacon 6 degrees of freedom graph search. Created all from scratch in C++, the Kevin Bacon 6 degrees is without question the most challenging project that I created while at UCSD. The end goal was to create a program that would take in as input, any 2 actors, and then return the shortest actor path between the two. That is-the path consists of consecutive actors that were in the same movie, continuing all the way until a path is found from the beginning actor to the ending actor. This project was extremely beneficial in teaching the importance of program/class design. In addition, I gained experience with file input/output, and constructing the world famous Dijkstra's shortest path algorithm from scratch.

Autocomplete Function

I truly gained a new level of understanding and respect after completing my own replica of an 'words suggestions' autocomplete function. This project takes as input a prefix of any word, and returns a list of valid words which themselves originate from a dictionary, that completes the given prefix. The data structure I used to create such a 'bank' of words was a multi trie tree structure. This tree was the clear preferred data structure, because the nature of the tree made it easier to branch out to another node, whenever a character was appended to a word.
